Imaging-based Spatial Transcriptomics (iST) vs Sequencing-based Spatial Transcriptomics (sST)

Spatial Transcriptomics (ST) technologies are generally classified into two primary categories: imaging-based and sequencing-based methods

Within the imaging-based category, technologies are further divided into in situ hybridization (ISH) and in situ sequencing (ISS)

Conversely, the sequencing-based category is primarily represented by in situ capturing (ISC) techniques

Imaging-based Spatial Transcriptomics (iST) identifies different mRNA species by targeting specific genes a priori using labeled complementary probes, which allows for the direct visualization of gene distribution within their native tissue context

Because of this targeted approach, imaging methods are often highly sensitive and capable of achieving subcellular or even single-molecule resolution

However, this reliance on specialized probes means that most imaging-based methods suffer from lower gene throughput and more limited gene panels than unbiased sequencing methods

Furthermore, as these techniques often require prior knowledge of the target genes to design the necessary labels, the total number of genes that can be measured simultaneously is restricted compared to transcriptome-wide approaches

Sequencing-based Spatial Transcriptomics (sST), primarily represented by in situ capturing (ISC), utilizes spatially barcoded arrays to capture polyadenylated RNA transcripts directly from tissue sections for analysis via next-generation sequencing (NGS)

Unlike imaging-based methods, sST is unbiased and provides transcriptome-wide coverage, meaning no a priori knowledge of target genes is required; this makes it ideal for discovery-driven research and identifying novel biological mechanisms or rare transcripts

While early iterations like the standard Visium platform are limited by a resolution of 55 µm diameter spots—which typically encompass a mixture of multiple cells—newer high-definition technologies such as Visium HD have significantly advanced this resolution

Visium HD reduces the spot diameter to 2 µm and eliminates gaps between spots, enabling gap-free and bias-free profiling at single-cell scale resolution

Other high-resolution sequencing methods include Stereo-seq, which achieves nanoscale resolution using 220 nm DNA nanoballs, and Slide-seq, which utilizes 10 µm barcoded beads

Despite these advancements in resolution, ISC technologies generally encounter lower transcript capture efficiency compared to imaging-based methods, as they often rely on the vertical diffusion of transcripts onto a capture surface rather than direct in-tissue analysis

Spatial Clustering and Domain Identification with Novae

Now we will use the Novae package to perform spatial clustering and domain identification.

One of the main steps in Novae is to compute the spatial neighbors. This is done using the spatial_neighbors function. This function requires the radius parameter, which is the radius of the neighborhood to be considered. For this particular cropped region, we will use a radius of 50.

Note also that connectivities need to be computed for each sample individually. If you have multiple samples in a single AnnData object, you can use the slide_key parameter to compute connectivities for each sample separately.

This radius parameter is important as too low of a radius will result in very sparse connectivities, missing important spatial relationships, while too high of a radius will start connecting cells that are not actually connected. See example of images below:

Cell Type Annotation using Enrichmap and Cell Type Gene Set Signatures

For cell type annotation, one can generally used methods that require a reference dataset, such as the new implementation of the RCTD method in python (check their repository), or gene set signatures from cell types that can be fetched from databases such as PanglaoDB, CellMarker 2.0 or trained models such as CellTypist and literature. For some diseases, it may be easier to find scRNA-seq datasets that may be used to perform cell type annotation, however, in rarer cancer types such as SCLC, this may not exist. Therefore, the best approach is to use cell type gene signatures and calculate scores to then annotate the segemnted cells and identified domains/niches/clusters.

We will now use the Enrichmap package to perform cell type annotation . (Talk more about this package).

We will start by simply filtering/making sure that our marker genes in our yaml file are present in our dataset. Then we run em.tl.score to perform the enrichment analysis and generate the scores.

Lets plot some of the scores to see how they look like:

If you compare these scores distributions to the markers from the original 10X tutorial, we can see that the scores follow similar distribution. We also see that there is an overlap between Tumor and Goblet cell scores.

To assign a cell type, one can then do a simple idxmax to assign the max scores for each cell ID to a given cell type. We can also use decoupler to run statistical tests and perform one-vs-rest comparisons between the scores of a given cell type in a given domain compared to all other domains in that sample.

We can then use these scores together with a elbow-gap method to assign cell types to each cell ID based on the compured scores. The reason why we can try this approach is because a given cell ID might still be capturing multiple cells (cell segmentation is working well, but is not perfect). If the scores are close to each other (e.g. the meanchanges from the ranking dataframe above) for a given domain, then we can assume that the cell ID is capturing multiple cells and we should assign it to the most enriched cell type(s).

Now lets plot the elbow plot

We see for example D999 is enriched for both Macrophages and Neutrophiles, although the meanchanges are quite small. With more data and cell types signatures (10+) we would be able to see more robust differences and assign a domain to 1 or more cell types based on this gap method using the meanchanges.
